# Tic Tac Toe

Petit guide simple pour le mini-jeu Tic Tac Toe.

## Fichiers
- `tictactoe.py` : contient le jeu et la boucle Pygame.


## Lancer le jeu individuel
1. Ouvrir un terminal.
2. Aller dans le dossier `snake_ladders/minigames/tictactoe`.
3. Exécuter : click sur run dans vs code 

## alterntive
- le jeu va être aleatoirement choisi dans le main jeu en executant `main.py` en clickent sur run.
 pour savoir plus veuillez lire le `README.md` principal


## Pré-requis
- Python 3.10 ou supérieur.
- `pygame-ce` installé.

## Installation de pygame-ce
```bash
pip install pygame-ce
```

## But du jeu
- Jouer contre l'ordinateur.
- Le joueur `X` commence.
- Le premier à aligner 3 symboles gagne.
- Si toutes les cases sont remplies sans gagnant, c'est une égalité.

## Notes
- Le dossier contient un mini-jeu autonome.
- Pour intégrer ce mini-jeu dans le projet principal, importer `TicTacToe` si la classe est définie dans `tictactoe.py`.
